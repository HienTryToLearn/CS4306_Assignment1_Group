Project Description 

This project implements the modified Gale-Shapley Stable Matching algorithm in Python. 
The program reads hospital and resident preference lists from an input file, assigns residents to hospitals with multiple available positions, 
verifies that the resulting assignment is stable, and outputs the final matching in the required format.
